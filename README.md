# Nicholas Realty

Next.js landing page for Nicholas Realty — Specialized Probate & Trust Brokerage.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Deploy (Vercel)

1. Push this folder to a new GitHub repo.
2. Go to [vercel.com](https://vercel.com) → **Add New** → **Project**.
3. Import the repo (leave **Root Directory** blank).
4. Click **Deploy**.

Every push to the main branch will deploy automatically.
